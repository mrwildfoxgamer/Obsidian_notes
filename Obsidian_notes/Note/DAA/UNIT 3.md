
## Searching

Searching is the process of finding a particular element in a list of elements 


### Methods of searching

1. linear search
2. Binary search


## Linear search

In linear search the algorithm checks each element one by one from the beginning of the list until the element is found.

`O(n)`-Time complexity

## Binary search
Binary search is divide and conquer algorithm used to search in a sorted array

`list must be sorted`

`n(log n)`-Time complexity



## Sorting
sorting means arranging data in a specific order, sorting converts unordered collection into an ordered collection 


### Bubble Sort
Bubble sort repeatedly compares adjacent elements and swaps them if they are in the wrong order

`O(n²)`

## Insertion sort
Insertion sort works like sorting playing cards in your hand,You take an element and insert it into the correct position in the sorted part

`O(n²)`-Time complexity
