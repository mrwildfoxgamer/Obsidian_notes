## Sorting Problems 
The sorting problem involves rearranging elements of a list in a specific order the Filed used to sort is called key eg `name` `id` etc

### Stable Sorting
A stable sorting algorithm keeps duplicating elements in the same order as the input `if there is a duplicate element then the 1st inputed elemnt is place 1st`

exaples of stable sorting
+ Bubble sort
+ insertion sort
+ merge sort
+ quick sort
`Sorting is one of the most fundamental problems in computer science and is widely used in databases, search engines, and operating systems`

## Algorithmic Efficiency 

Algorithmic efficiency calculates the amount of time and resource used by an algorithm to solve a problem

#### Time Complexity
Time required for the algorithm to run expressed  using **Big-O notation**

#### Space Complexity
Amount of memory used by the algorithm

Factors include 
+ Input storage
+ Temporary variables
+ Data structures

`Efficient algoritham aims to minimize both time and memory usage`
