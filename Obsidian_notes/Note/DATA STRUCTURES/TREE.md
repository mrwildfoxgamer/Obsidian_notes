[[DSA]]

* Binary tree is a non leaner two dimensional data structure 
* A tree contains a finite amount of nodes connected together in a parent child hierarchical order 
* There is no restriction on the number of child nodes
### Application of trees

- Binary search tree - used to find an element 
- Tries - A modified version of tree that is used to  store information on Routers 
- Syntax Tress - used by the compiler to validate Syntax of every programs
