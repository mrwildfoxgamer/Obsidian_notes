## Static Web page

*The content of the web page stays the same each time the web page is loaded*
- *They are easy and cheap to develop and host*
- Requires web development skills to update the site not as useful for the user
- The source code should be edited in order to make updates

## Dynamic Web page
*The content of the web page changes with new information*

- *Much  easier to update can work in such a way that users can collaborate*
- Slower and more expensive to develop and hosting is more expensive

## CSS
* *cascading style sheet*
* front end programming language
* There are 2 ways to insert CSS:
	* **Internal** The internal style is defined inside the style element  inside the head section
	* **external** The CSS code is written in different file with .css extension 
	* **Inline** Add the style attribute to the element The style attribute can contain any CSS property
* CSS selectors are of two types
	* **Id selector**: the id of an element is unique with in page `#`  is used before the id of an element
	* **class selector** : It select the elements with a specific class attribut `.` is used before the class name
